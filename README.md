## üìñ Description

Fills the crates of the various structures in Minecraft with modified vanilla loot to find even more unique and special items!

-----

#### Features:
- adds no new items and blocks! That means if you delete the mod later, you have still all custom items!
- vanilla items with their own name, lore, enchantments, attributes and components
- the loot spawns in many different structures, matching the theme

-----

#### Create or change your own loot:
<details>
<summary>Spoiler</summary>

You can add or delete your own items to the game! You must only modify the `loot_table.json` files in the Mod. (Only for advanced users!)

Or simply change the spawn values or the stats of the items.

-----

#### With datapack (for newer versions):

<details>
<summary>Spoiler</summary>

Use this basic datapack and modify it:

- [1.19.x](https://www.curseforge.com/minecraft/mc-mods/customized-dungeon-loot/files/3878801)

</details>


-----

#### With editing of the mod files itself:

<details>
<summary>Spoiler</summary>

- Open the mod archive (Customized Dungeon Loot.jar)

- (**in 1.12**) Go in the folder: `assets\customizeddungeonloot\loot_tables`

(**in 1.14 and higher**) Go in the folder: `data\minecraft\loot_tables`

- And change the `loot_table.json` file that you need.

</details>

-----

#### If you don't know, how to edit loot_tables, here is a little help for you:

- https://misode.github.io/loot-table/

- http://minecraft.gamepedia.com/Loot_table

- http://www.planetminecraft.com/mod/lds-loottable-generator/

</details>
